Basic Syntax in TypeScript

Learn the fundamental rules and structure for writing clean TypeScript code

Understanding Syntax

Learning TypeScript syntax is like learning the grammar of a new language. Variables, functions, and classes are like words and sentences, while syntax rules act like grammar that keeps communication clear.

Building Blocks of TypeScript

Variables

Store values using types like string, number, and boolean.

Functions

Reusable blocks of code that perform specific tasks.

Classes

Blueprints used to create objects and organize logic.

TypeScript Program Structure

A TypeScript program usually contains:

  • Variables – store information
  • Functions – perform tasks
  • Classes – organize behavior
  • Statements & Expressions – instructions executed by the program
  • Comments – notes that explain the code

Code Example


// Variable
let productName: string = "Bananas";

// Function
function printProduct(name: string): void {
  console.log("Product: " + name);
}

// Class
class Store {
  open(): void {
    console.log("Store is open!");
  }
}

// Using the function
printProduct(productName);

// Using the class
let store = new Store();
store.open();
  • let productName: string declares a variable.
  • function printProduct() defines a reusable function.
  • class Store defines a blueprint for objects.
  • new Store() creates an object from the class.

Identifier Rules

Identifiers are names for variables, functions, and classes.

  • Can include letters, numbers, _ or $
  • Cannot start with a number
  • Cannot contain spaces
  • Cannot be a reserved keyword
  • Are case-sensitive

Compiling and Running TypeScript


1. Save the file
   example.ts

2. Compile it
   tsc example.ts

3. Run the JavaScript
   node example.js

Useful Compiler Flags


tsc --help

tsc --removeComments

tsc --watch
