Code Example


let name: string = "V S I Raju";
let age: number = 22;

let message: string = `Hello ${name}! You are ${age} years old.`;

console.log(message);
  • let name: string declares a variable that must always store text.
  • let age: number declares a variable that must store a number.
  • `Hello ${name}! You are ${age} years old.` is a template string that inserts variables directly into text.
  • console.log(message) prints the final message to the console.

Try changing the values of name and age and run it again!

Type Safety Example

One of the biggest advantages of TypeScript is static typing. The compiler checks for mistakes before the program even runs. 


let age: number = 22;

age = "twenty two"; 
// ❌ TypeScript Error:
// Type 'string' is not assignable to type 'number'

JavaScript would allow this mistake, but TypeScript catches it early, making your programs more reliable.

Where Is TypeScript Used?

TypeScript is widely used across modern software development.

  • Web Applications — React, Angular, and modern frontend frameworks
  • Backend Development — Node.js APIs and server applications
  • Mobile Applications — Ionic and React Native apps
  • Desktop Software — Electron applications
  • Game Development — Browser based games and engines

Getting Started with TypeScript

To begin using TypeScript, you only need a few simple tools.


1. Install Node.js

2. Install TypeScript
   npm install -g typescript

3. Compile TypeScript
   tsc app.ts

4. Run the generated JavaScript
   node app.js

TypeScript code is compiled into JavaScript, which can run in any browser or JavaScript runtime.

Key Takeaways

  • TypeScript is a typed superset of JavaScript.
  • It helps detect errors early during development.
  • It improves maintainability and scalability.
